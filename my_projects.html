Personal Projects

Breakout Room Generator

This program is something I created at the very beginning of COVID, when we had to move from in-person classes and gatherings to Zoom meetings. I created this program that efficiently generates breakout rooms for a Zoom meeting. These breakout rooms maximizes the number of new interaction between the participants and creates desired breakout rooms for the Zoom host.

Sudoku Solver

From a young age, I was interested in Sudoku and solving the puzzle using generic strategies that were taught in the little booklets that I bought from the bookstore. Therefore, I decided to try to convert these strategies into code and see if I could write a program that could use the same techniques I used to solve any given sudoku puzzle. This program could solve all easy to expert sudoku puzzles using basic solving techniques.

Game of Life

John Conway's Game of Life is a cellular automaton introduced by the British mathematician John Horton Conway in the 1970s. It's a simple computer game that requires no interaction from the user and simply a game that plays itself using different conditions that are preset. This game is extremely interesting and creative, so check out how the game works by clicking the hyperlink to this project!
