Game of Life

Source Code (Github)

Reason behind writing this program:

Conway's Game of Life was a game I was introduced to during my AP Computer Science class as an interesting game that we should explore in our own time if interested. Therefore, I spent some time looking into this game and was really fascinated by how John Conway was able to come up with this zero player game were the "cells" just keep dying and regenerating to create new patterns.

How this program works:

This program has three rules that has to be followed each iteration:

1. Any shaded box with fewer than two neighboring shaded boxes is unshaded
2. Any shaded box with more than 3 neighboring shaded neighbors is unshaded
3. Any unshaded box with exactly 3 neighboring shaded boxes is shaded
